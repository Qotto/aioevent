.. _installation:

============
Installation
============

Latest Release
--------------

Install with your favorite package manager

With Pip:

.. code-block:: bash
   :linenos:

   pip install tonga

With Pipenv:

.. code-block:: bash
   :linenos:

   pipenv install tonga

Releases are also listed at https://github.com/Qotto/tonga/releases

Bleeding-Edge
-------------

.. code-block:: bash
   :linenos:

   git clone https://github.com/Qotto/tonga
   pip install ./tonga
